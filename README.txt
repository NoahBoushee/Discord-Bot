A generic discord bot with administrative functions as well as other functions.

Please ensure that the config.json is filled out fully otherwise the bot will not function correctly
	prefix - the prefix for the bot commands
	adminRole - the name of your discord servers admin role
	token - discord application token
	giphyToken - your giphy token
	youtubeAPI - youtubeAPI token
	prefix2 - the second prefix for the bot, this is for database relations
	serverID - ID of your server, you can get it by right-clicking the server name
	allGeneralChannel - Id of the general channel
	errorDumpChannel - the bot spam channel
	defaultRole - the defaultRole that is assigned to all users
	noPornRole - the role to hide the NSFW channel if one exists
